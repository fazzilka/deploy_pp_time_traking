# Проект обучающегося (Митин Олег Дмитриевич БПИ2402)

## Назначение

Проект предназначен для учёта задач и интервалов времени, затраченного на выполнение задач, с автоматическим вычислением суммарного времени. Система позволяет создавать задачи, запускать и останавливать измерение времени, хранить историю интервалов времени и получать сводку по общему времени и задачам с наибольшими затратами времени.

## Проект развёрнут на сервере, подключён CI/CD, метрики и логи отображаются в Grafana.

### ссылка на развёрнутый проект: https://time-tracking.online/

## Документация

Документы находятся в каталоге `docs/`.

## Запуск и использование

Тут будет инструкция по запуску и работе с `make`.
## Запуск серверной части проекта

Запуск выполняется из папки `server/`.

1. Перейдите в папку серверной части проекта: `cd server`.
2. Запустите все сервисы одной командой: `make run`.
3. Проверьте журнал работы серверной части проекта: `make logs`.
4. Для остановки сервисов выполните команду: `make down`.

## Документация и наблюдаемость

Документация находится в каталоге `docs/` в корне репозитория.

- Описание программного интерфейса приложения: `docs/Описание программного интерфейса приложения.md`
- Описание предметной области: `docs/01_Описание_предметной_области.md`

После запуска сервиса доступны:
- Программный интерфейс серверной части и автоматическая документация: `http://localhost:8000/docs`
- Метрики для системы сбора метрик: `http://localhost:8000/metrics`
- Система сбора метрик: `http://localhost:9090`
- Система визуализации: `http://localhost:3000` (имя пользователя `admin`, пароль `admin`)
- Журналы приложений доступны в системе визуализации через источник данных Loki и в терминале через `cd server && make logs`
