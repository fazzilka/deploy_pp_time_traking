# Отчет о тестировании

## 1. Назначение

Отчет о тестировании фиксирует результаты выполнения сценариев тестирования системы учёта задач и интервалов времени, а также информацию о выявленных и исправленных дефектах.

## 2. Среда тестирования

Тестирование выполнялось в локальной среде разработки при запуске приложения в контейнерах.

Используемые компоненты:
- серверная часть приложения;
- база данных;
- окружение контейнеризации.

## 4. Выявленные дефекты и исправления

### Дефект 1. Повторный запуск таймера создавал второй активный интервал времени
**Проявление:**
при повторном запуске измерения времени система могла создать второй активный интервал времени.

**Ожидаемое поведение:**
у одной задачи должен существовать не более одного активного интервала времени.

**Исправление:**
добавлена проверка наличия активного интервала времени и обработка конфликта состояния.

---

### Дефект 2. Остановка таймера без активного интервала времени приводила к некорректному ответу
**Проявление:**
при остановке измерения времени без активного интервала времени система могла возвращать успешный результат без изменений.

**Ожидаемое поведение:**
операция должна быть отклонена как конфликт состояния.

**Исправление:**
добавлена проверка отсутствия активного интервала времени и корректный ответ об ошибке.

---

### Дефект 3. Суммарное время не пересчитывалось после закрытия интервала времени
**Проявление:**
после остановки измерения времени суммарное время могло не измениться.

**Ожидаемое поведение:**
суммарное время должно увеличиваться на длительность закрытого интервала времени.

**Исправление:**
добавлен пересчёт суммарного времени после закрытия интервала времени.

## 5. Итог

Все предусмотренные планом тестирования сценарии выполнены успешно. Выявленные дефекты исправлены. Система готова к демонстрации.
